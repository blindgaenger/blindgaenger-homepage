Compile and install Monit

Find the latest stable Monit at http://www.tildeslash.com/monit/download/

Download and unpack {{bash $ cd /tmp $ wget http://www.tildeslash.com/monit/dist/monit-4.10.1.tar.gz $ tar -xvzf monit-4.10.1.tar.gz $ cd monit-4.10.1 }}
