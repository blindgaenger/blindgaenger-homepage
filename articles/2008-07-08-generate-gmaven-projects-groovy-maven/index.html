Generate GMaven projects (Groovy & Maven)

This is more like a cheatsheet for me. But I’ve used the original GMaven docs if you miss something.

Start generation from the project folder using maven itself:

{% highlight bash %} mvn archetype:generate -DarchetypeGroupId=org.codehaus.groovy.maven.archetypes -DarchetypeArtifactId=gmaven-archetype-basic {% endhighlight %}
